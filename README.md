Government Engineering College Palamu Website

A modern and responsive college website built for Government Engineering College Palamu using React, Tailwind CSS, and modern frontend technologies.
The project provides students and faculty with an interactive platform to access club information, announcements, assignments, events, and campus resources.

🚀 Features
🎓 Modern College Landing Page
📢 Dynamic Announcement Section
🏛️ Clubs & Communities Showcase
🖼️ Responsive Portrait/Landscape Image Handling
📂 Assignment Management UI
✅ Faculty Assignment Approval/Rejection Status
🔥 Firebase Integration
⚡ Fast & Responsive UI
🌙 Clean and Attractive Design
📱 Fully Mobile Responsive
🛠️ Tech Stack
React.js
Tailwind CSS
JavaScript
Firebase
Vite
